# 小资金进攻交易系统升级

Last updated: 2026-05-10

本文用于把“小资金做大”的交易系统从研究框架升级为执行框架。核心不是更努力预测，而是先定义风险，再决定是否下注。

## 1. 五个核心问题

### 1.1 错的时候亏多少

先定义 `1R`，再定义仓位。

公式：

```text
可买仓位比例 = 单笔允许亏损比例 / 从买入价到结构止损位的跌幅
打到止损的账户亏损 = 个股仓位比例 × 止损跌幅
```

例子：云南锗业截图价 86.07，结构止损参考 79.50，止损距离约 7.63%。若持仓 64.55%，打到止损约亏账户 4.93%，已经接近当前单笔 5% 风险预算。

结论：重仓不一定错，但重仓必须配窄止损、强流动性、强板块和严格执行。没有结构止损的交易，不允许加仓。

### 1.2 对的时候能不能放大

放大只能发生在三种情况：

1. 原仓已经盈利，且止损可以上移到不伤本金或小亏。
2. 市场状态仍允许进攻，板块龙头没有退潮。
3. 个股回踩不破关键位，或突破后承接强于预期。

禁止把“浮亏摊低成本”伪装成放大。真正的放大是用优势和利润滚动，不是用希望补仓。

### 1.2A 阶段性资金核心不能因为高位机械卖飞

2026-05-11 复盘新增，2026-05-12 再校正：对当前市场资金正在追捧、且有成交、宽度、承接、反馈和赚钱效应验证的阶段性核心、龙头、中军和高辨识度趋势票，`高位` 只代表止损距离、波动和仓位风险，不再单独构成卖出理由。任何方向都不能被静态定义为永恒主线，必须每天用资金行为重新验证。

1. 原始预期是否继续增强：产业逻辑、政策、订单、价格、业绩、公告和板块映射有没有新证据。
2. 情绪是否继续增强：连板高度、核心票承接、昨日强势股反馈、炸板修复、亏钱效应是否可控。
3. 是否有人继续相信预期并买入：成交额、换手、主力/超大单代理、融资余额和融资买入变化、港股通/基金持仓变化，能取到则必须检查；取不到要标缺失。
4. 板块是否共振：当前资金追捧方向里的龙头、中军、补涨、低位扩散是否同步，不能只看单股涨幅；如果资金已经切换，原方向立即降级。
5. 失效条件是否出现：放量滞涨、长上影、跌破关键均线/筹码区、融资盘快速兑现、龙头退潮、板块宽度收缩。

阶段性资金核心减仓必须优先使用分批退出和回补机制：

| 动作 | 规则 |
| --- | --- |
| 先减一半 | 当仓位过重、波动超出账户承受、但逻辑未证伪时，只减风险，不清空核心。 |
| 留核心仓 | 只要预期、情绪、成交和板块共振仍在，保留可继续吃主升的底仓。 |
| 设回补条件 | 重新站上关键位、板块共振、成交继续放大、融资/资金承接不恶化时，允许买回。 |
| 不乱换弱票 | 只有新票的预期强度、板块地位、流动性和风险收益明显优于原核心，才允许换仓。 |

执行约束：如果只是因为“涨高了”而建议清仓，必须被视为低质量结论；如果逻辑未坏但仓位过大，标准动作是减仓、上移止损、留底仓和设置回补条件。

### 1.3 连错几次会不会停手

硬规则：

1. 单日亏损达到总资金 2%：停止新开仓，只处理持仓。
2. 单周亏损达到总资金 5%：仓位上限降到 50%，直到重新出现高胜率环境。
3. 连续 3 笔按计划止损：暂停追涨，只允许低吸核心或空仓等待。
4. 当天出现非计划交易：停止新增交易，收盘复盘错误来源。

小资金做大的关键不是不亏，而是避免连续情绪化亏损。

### 1.4 是否只在最有优势的环境下注

每天先判市场状态，再判断个股。市场状态决定仓位权限，个股只决定买谁和怎么买。

#### 强进攻日

特征：

1. 指数和情绪同向修复或上攻，成交额放大。
2. 涨多跌少，涨停数量扩大，跌停和核按钮较少。
3. 昨日涨停反馈好，连板高度提升或高标承接强。
4. 当前资金追捧方向清晰，龙头、中军、补涨形成梯队。
5. 竞价阶段核心票超预期，开盘后资金继续回流。

打法：允许重仓核心，但只做龙头、中军、已验证趋势票；弱跟风仍然不能重仓。

#### 轮动日

特征：

1. 指数不差，但资金追捧方向持续性不足。
2. 板块快速切换，昨天强的今天容易分化。
3. 龙头还能活，但跟风持续性弱。
4. 高开容易兑现，追高胜率下降。

打法：只低吸，不追高；优先强换弱和持仓管理。总仓位应低于强进攻日。

#### 退潮日

特征：

1. 指数走弱或成交萎缩，涨少跌多。
2. 跌停、炸板、断板负反馈增加。
3. 高标补跌，昨日强势股低开低走。
4. 板块龙头承接失败，中军也不稳。
5. 好消息不涨，坏消息放大。

打法：只处理持仓，不开新仓；短线票反弹先减，破位先走。

#### 冰点修复日

特征：

1. 前一阶段已经恐慌，亏钱效应充分释放。
2. 跌停减少，核按钮缓和，高辨识度个股开始止跌。
3. 指数未必强，但情绪核心开始反包或低位启动。
4. 板块尚未全面扩散，需要确认修复是否能延续。

打法：只做核心反包或低位首启，小仓试错，失败快撤。不能把修复当成新主升。

#### 混沌日

特征：

1. 指数、情绪、板块互相矛盾。
2. 龙头强但跟风弱，或板块强但指数拖累。
3. 成交额不足，资金东拉西扯。
4. 交易机会看起来很多，但持续性差。

打法：降低仓位，等待确认。混沌日最大的风险是为了交易而交易。

## 2. 09:28 到 09:35 的判断流程

上午只保留 09:28 竞价校验。09:28 是第一道可交易过滤，09:30-09:35 是开盘承接确认；不要用收盘后才知道的数据追当天的后排机会。

| 时间 | 状态名称 | 可靠度 | 主要证据 | 交易权限 |
| --- | --- | ---: | --- | --- |
| 09:25-09:28 | 竞价状态 | 中 | 龙头/中军/持仓竞价，昨日强势股反馈，高标是否被核，板块竞价广度 | 决定是否允许开仓或加仓 |
| 09:30-09:35 | 开盘确认状态 | 中到高 | 高开承接、低开修复、放量方向、板块扩散、核心票分时强弱 | 执行、取消或只小仓试错 |
| 10:00以后 | 盘中确认状态 | 高 | 当前资金追捧方向是否持续、成交是否兑现、后排是否掉队、指数是否拖累 | 管理持仓、等核心回踩，少追后排 |
| 收盘后 | 复盘状态 | 最高 | 全天结构、龙虎榜、资金流、公告补充、交易结果 | 更新系统，不用于追当天 |

原则：用小仓买不确定，用加仓买确认，用止损处理错误。

### 09:28 竞价校验

用竞价建立当天第一版交易计划：

1. 龙头是否强于预期。
2. 中军是否同步。
3. 跟风是否虚强。
4. 持仓是否高开兑现、低开恐慌或平开弱承接。
5. 是否触发取消交易条件。

竞价比盘前更重要，但仍不是最终答案。竞价强但开盘不能承接，计划要降级；竞价弱但开盘快速收回关键位，可以从取消改为观察。

### 09:35 首个五分钟

只确认，不冲动：

1. 高开是否能承接。
2. 低开是否能快速收回关键位。
3. 放量上涨是否有板块配合。
4. 计划外机会是否真的比持仓更优。

如果 09:35 前仍无法确认资金追捧方向和承接，今天就按轮动或混沌处理，不要为了错过感去追后排。

## 3. 四类标准打法

### 阶段性资金核心强趋势

只在强进攻日或冰点修复转强后使用。买点是突破确认或缩量回踩后转强；加仓只能用利润滚动；退潮、放量滞涨、龙头失速时降仓。

### 轮动低吸核心

适合轮动日。只买核心票回踩支撑，不追高。低吸后如果不能快速脱离成本区，说明资金承接不足。

### 情绪补涨/反包

适合情绪强或冰点修复初期。仓位要小，速度要快。最高标崩、龙头跳水、炸板弱回封时退出。

### 持仓防守与利润保护

适合退潮日和混沌日。核心目标不是赚更多，而是不把已经赚到的利润和本金暴露给低胜率环境。

## 4. 交易日志反馈

每笔交易收盘后必须记录：

1. 当天市场状态。
2. 使用的打法。
3. 计划动作和实际动作是否一致。
4. 计划亏损是多少 R。
5. 实际结果是多少 R。
6. 错误来自市场判断、板块判断、个股选择、买点、卖点、仓位还是执行。
7. 下一笔仓位应该上调、维持还是下调。

没有交易日志，系统只会变复杂，不会稳定进步。

## 5. 【补充】最容易忽略的地方

1. 小资金做大要进攻，但进攻不是每天满仓。
2. 龙头能重仓，跟风不能重仓。
3. 浮亏票的第一任务是重新证明自己，而不是等它回本。
4. 大赚后第二天最容易扩大风险，必须按市场状态重新评估。
5. 仓位不是信心表达，仓位是止损距离、胜率、赔率和流动性的结果。
6. 不把任何方向永久写死为主线；每天重看成交、宽度、承接、反馈和赚钱效应，用“当前资金更偏向什么”替代“某某一定是主线”。
